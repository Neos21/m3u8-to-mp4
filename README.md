# m3u8-to-mp4

.m3u8 ファイルを元に .mp4 ファイルに結合して書き出すスクリプト。


## `m3u8-to-mp4.bat`

このバッチファイルに `.m3u8` ファイルをドラッグ・アンド・ドロップすると、このバッチファイルがあるフォルダに `.mp4` ファイルを書き出す。

__要 ffmpeg。__ `$ choco install ffmpeg` などでインストールしておくこと。
